Wandrr Frontend Backend

Pitched, developed, and deployed alongside my team for our project "Wandrr" a full stack travel and map application built on a React frontend and Django REST API backend managing a GAL-compatabile PSQL database, integrated with Mapbox and Wikipedia's APIs.

Wandrr is an app that uses the mapbox API and either the user's GPS coordinates or point of focus on the map to query other APIs about historical and interesting information associated with addresses or GPS coordinates close to where the user is. The backend automatically caches previous queries for very fast response times in areas of frequent use.
